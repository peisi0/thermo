## 📱 Projects Overview

### 1. **OurChat** - WeChat Clone
**Path:** `/ourchat`

A pixel-perfect recreation of WeChat's mobile interface with dark mode support and AI chatbot integration.

**Features:**
- 🌙 Native dark theme design
- 🤖 Rule-based AI chatbot with context memory
- 📱 Mobile-first, touch-optimized interface
- 🔄 Smooth tab switching (SPA feel)
- 🔍 Real-time search functionality
- 📦 PWA support for installation

**Tech Stack:** HTML5, CSS3 (Flexbox), Vanilla JS

[**Launch Demo →**](https://peisi0.github.io/ourchat/) | [**Read Docs →**](https://peisi0.github.io/ourchat/README.md)

### 2. **Neon Thermometer** - Interactive Temperature Display
**Path:** `/thermometer`

A stunning interactive thermometer with neon glow effects and real-time visual feedback.

**Features:**
- 🌡️ Temperature range: -30°C to 50°C
- 🎨 Dynamic color transitions (blue → red)
- ✨ Animated bubble effects
- 📏 Dual scale display (Celsius/Fahrenheit)
- 🖱️ Drag-to-adjust interaction
- 💡 Ambient lighting effects

**Tech Stack:** CSS Variables, CSS Animations, Canvas API

[**Launch Demo →**](https://peisi0.github.io/thermometer/)

### 3. **Smart App Banner** - Multi-App Deep Linking

## 🚀 Getting Started

### Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/peisi0/fun_web.git
   cd fun_web
   ```

2. **Open in browser:**
   ```bash
   # Open the main index
   open index.html

   # Or open individual projects
   open ourchat/index.html
   open thermometer/index.html
   ```

3. **Or use a local server:**
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js
   npx serve

   # PHP
   php -S localhost:8000
   ```

   Then visit `http://localhost:8000`

### No Build Required! 🎉

All projects run directly in the browser without any build step, transpilation, or dependencies.

## 🎨 Design Philosophy

### Why Vanilla?

- **🚀 Zero Build Time:** Instant refresh, no compilation
- **📦 No Dependencies:** No node_modules, no package.json
- **🎯 Learning Focused:** Understand core web APIs
- **⚡ Performance:** Minimal bundle size, fast load times
- **🔧 Maintainability:** Simple, readable code

### Mobile-First Approach

All projects are designed with mobile devices in mind:
- Touch-optimized interactions
- Responsive layouts
- Viewport optimization
- Performance considerations

## 💡 Key Learnings

### Techniques Demonstrated

1. **Single Page Applications (SPA)**
   - Tab switching without page reload
   - Dynamic content loading
   - State management with vanilla JS

2. **CSS Animations & Transitions**
   - Smooth UI transitions
   - Keyframe animations
   - Transform effects

3. **Mobile-First Design**
   - Flexible layouts
   - Touch event handling
   - Viewport management

4. **Progressive Web Apps (PWA)**
   - Service Workers
   - Manifest files
   - Offline functionality

5. **Interactive Visualizations**
   - Real-time data representation
   - Dynamic color mapping
   - Particle systems
